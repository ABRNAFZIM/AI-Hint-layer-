# Contributing Guidelines

Thank you for your interest in contributing to the **AI Scaffolding Hint Layer** project. This project explores responsible AI, adaptive scaffolding, and human‑centred learning design. Contributions of all kinds are welcome.

---

##  Ways to Contribute

- Improve documentation
- Add examples or scenarios
- Suggest enhancements to the architecture
- Propose new frustration signals
- Improve pseudocode clarity
- Share research or pedagogical insights
- Report issues or inconsistencies

---

## Development Guidelines

- Keep contributions aligned with responsible‑AI principles.
- Avoid adding code that gives full solutions to student tasks.
- Maintain academic integrity guardrails.
- Ensure all hints remain scaffolding‑based.
- Document any new components clearly.

---

## Responsible AI Expectations

All contributions must:

- Support student wellbeing  
- Avoid harmful or misleading behaviour  
- Respect human‑in‑the‑loop design  
- Promote learning, not shortcuts  

---

## Licensing

By contributing, you agree that your contributions will be licensed under the **MIT License**, consistent with the rest of the project.

---

## Getting Started

1. Fork the repository  
2. Create a new branch  
3. Make your changes  
4. Submit a pull request with a clear description  

Thank you for helping build a responsible, human‑centred AI learning system.
